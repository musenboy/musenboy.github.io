发表于 2016-07-16

GitHub是什么?

  • GitHub是基于git实现的代码托管，git可能是目前最先进的版本控制系统。
  • GitHub非常强大，它允许用户跟踪其他用户，组织，软件库的动态，对软件代码的改动和bug提出评论等。
  • GitHub上有全世界大牛的开源代码，可以相互交流，开阔眼界，提升编码水平。
  • GitHub免费代码托管，只有私人仓库才需要收费，不用担心代码丢失，自己管理，保存可靠。

添加SSH Key到GitHub

在本机设置SSHKey之后，需要添加到GitHub上，以完成SSH链接的设置。

打开本地id_rsa.pub文件。此文件里面内容为刚才生成密钥。

准确的复制这个文件的内容，才能保证设置的成功。登陆github系统。点击右上角的 Settings->Personal settings—>SSH keys —> New SSH Key 把你本地生成的密钥复制到里面（key文本框中），点击Add SSH key就可以添加完成。
